# Fullbright Mod

 Enables Fullbright funcionalty for higher versions of Minecraft using a workaround. (fabric only)
 
 Requires [Cloth Config API](https://modrinth.com/mod/cloth-config) and [Mod Menu](https://modrinth.com/mod/modmenu) (does not apply for the beta version for fabric-1.20)
